There are 6 main goals for creating great User Interfaces, and that gives us 6 main criteria to analyze the user interfaces of different websites and applications.
This survey is intended to get your opinion on the user interface of the Tumblr main page, using these 6 criteria. Here is a brief description of each of the criteria.

  • Proper Functionality: This criteria refers to the systems funtions, and whether or not they function as expected, based on the system specification and the users expectations.
  • Consistency: This criteria asks to what extent all the UI elements have the same type of behavior throughout the system. For example, clicking on a particular button that appears throughout the system should always have the same result.
  • Efficiency: This criteria judges how well the UI 'helps' the user complete taks in a timely manner, with no extra steps.
  • Usability: This criteria is based on the intuitiveness of the UI, how well a user can use the system without training.
  • Reliability: This criteria measures the how the system functions without major flaws or errors.
  • Security and Data integrity: This criteria discusses how well the system protects the users data. This is often difficult to determine using only the UI.
